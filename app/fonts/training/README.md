Training page local fonts

These font files are self-hosted for the training-page sentence font switcher.

Families included:
- Newsreader
- Public Sans
- Plus Jakarta Sans
- Exo 2

Source:
- Downloaded from Google Fonts / fonts.gstatic static files for local self-hosting

Use:
- Loaded only through `next/font/local` in `lib/training-fonts.ts`
- Scope is limited to the training-page English sentence font selector
